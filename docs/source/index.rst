Bem vindo a documentaçao da plataforma Vivace!
==================================

Bem vindo ao repositorio da Vivace!

Vivace é um projeto de plataforma de aquisiçao de dados e telemetria
desenvolvido durante o ano de 2016 pela equipe Céu Azul Aeronaves,
participante da competiçao anual SAE Aerodesign.

Todo o codigo aqui presente é de autoria dos membros da equipe de
telemetria de 2016, exceto quando explicitamente avisado.

Este repositorio se encontra atualmente em fase de limpeza, organizaçao
e documentaçao, a fim de tornar o codigo reutilizavel e facilitar
contribuiçoes futuras.

Nosso objetivo é uma plataforma de aquisiçao de dados aberta, entao
sinta-se livre para fazer parte deste projeto, seja voce da Céu Azul
ou nao!
